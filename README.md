# Learnagochi

Learnagochi is a gamified learning experience that blends a virtual companion with short learning quests. You progress by completing tasks, earning rewards, and unlocking new scenes, all designed to make practice feel playful and consistent.

## What You Can Do
- Explore story-driven scenes and a cozy UI
- Complete bite-sized learning activities and quests
- Collect rewards and track your progress

## How To Run
Open `index.html` in a browser. The main game pages are also available in the `Game` and `Final` folders.
